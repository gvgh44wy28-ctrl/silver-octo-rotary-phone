3D Deck Model — CAD Editor

The overall deck outline and every staircase's position/orientation are traced from your 2D Plan Designer drawing — draw as many Stair items as you need there (even on different sides), and Rebuild picks all of them up here, each with its own opening cut into the railing. Counts and sizes (boards, joists, risers, posts, rail spacing) still come from the calculators. Every object here — the platform, each post, joist, rail, and stair step — is its own editable piece afterward. Click an object (or pick it in the Outliner) to set its exact size, position, and rotation in the Inspector. Switch Model Units to inches for finer control on small parts. Perimeter railing builds real 4x4 rail posts at the spacing set in Railing Layout (4/5/6 ft OC), with a standard 5.5in rail cap and infill balusters between posts at the calculated baluster spacing. Corners get a distinct post on each side, set back slightly so the rail caps run past them and meet cleanly at 90°. Railing posts can be angled from the Railing Layout tab — the tilt pivots from the post's base so it's actually visible. Any rail or stair handrail segment can be stretched or shrunk by dragging once Resize mode is on; resizing only ever changes length, never width.

Help Center

Quick guide for using SpanWright Decks.

1. Start on the Dashboard

View the deck model, takeoff summary, AI recommendations, and live pricing. The tile buttons are all interactive.

2. Design the Plan

Use the 2D Plan Designer to draw an accurate footprint, then check it out in the 3D Model tab.

3. Run the Calculators

Deck, Railing, Stairs, and Materials & Fasteners all feed a single running estimate.

4. Review Board Pricing

Compare wood versus composite trends; the current month updates live every few seconds.

5. Build the Estimate

Fill in customer/company details on Estimate & PDF, attach photos, then Generate PDF / Save.

6. Save & Print

Save Project stores a full snapshot in Saved Estimates for this session; Print Installer Step List gives your crew a work-order printout.
