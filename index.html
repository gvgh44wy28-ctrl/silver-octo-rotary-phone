Help Center

Quick guide for using SpanWright v2.0.

1. Start on the Dashboard

Use the dashboard to view the deck model, takeoff summary, AI recommendations, live pricing, and project stats.

2. Design the Deck

Open Designer to move deck, stair, and railing pieces around the grid. Future versions can connect these objects directly to material takeoffs.

3. Build an Estimate

Enter customer details, dimensions, material type, and labor rate. The estimate updates automatically.

4. Review Board Pricing

Use the pricing graph to compare wood versus composite board trends. Later this can be tied to supplier APIs.

5. Save Projects

Tap Save Project to store the current project snapshot in the Saved Files folder.

6. Export PDF

Tap Download PDF to open the print dialog. On iPhone, choose Share/Print and save the estimate as a PDF.
